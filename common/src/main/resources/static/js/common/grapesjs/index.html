    イベント

    オフラインイベント（セミナー、レッスンなども含む）の企画・集客・運営で発生する作業全般を、網羅的に管理できるメニューです。これによって、オフラインイベントの開催を、収益管理しつつ効率よく実施することができます。

    まずは、「イベント企画」で開催するイベントの内容を具体化し、イベントを告知してください。その後「イベント運営」で随時状況を確認しながら適切なアクションを取ります。また、イベント当日の受付業務にも便利に使えます。

    オンラインショップ

    オンラインショップを開設するメニューです。これを利用することで、テナント上にフル機能を持つオンラインショップを簡単に加えることができます。

    掲載商品を「商品管理」でJimzen上に登録するところから始めましょう。その後注文に関するルールを設定。オンラインショップのデザインに関しては「Webサイトデザイン」メニューでドラッグ＆ドロップ形式で行います。Jimzenでは、注文以降の手続きに沿った「注文管理」も容易に行えるため、テナントオーナーはスムーズに販売活動を展開できます。

    会計

    テナント活動で生じる収支を管理するための集計メニューです。Jimzen利用料、イベント、コース、オンラインショップなどの活動別、決済方法別にリアルタイムベースで売り上げを把握することができます。また、未領収や未返金などのステータス管理も随時行えるため、迅速に次のアクションに移れます。さらに、特定の目的で会計情報を検索することもできます。

    ここには特にテナントオーナーが設定する項目はありません。

    共通設定

    Jimzen上で利用する多様なリソースを設定・登録しておくメニューです。ここで設定した情報や文章、フォーマットがテナント活動のさまざまな場面で反映されます。このメニューをうまく利用することによって、Jimzenをテナントの属性に合わせてカスタマイズでき、首尾一貫したテナントを作りあげることができます。

    サブメニューは豊富です。まずは、マスター情報を登録し、テナントメンバー向けの登録フォームを作成するところからスタートしてください。
